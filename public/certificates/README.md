# Certificate PDFs

This folder should contain your certificate PDFs. Add the following files:

1. **fullstack-engineer.pdf** - Full-Stack Engineer Certificate (150 hours)
2. **ai-data-foundations.pdf** - AI & Data Foundations Certificate (39 hours)
3. **python3.pdf** - Python 3 Certificate (24 hours)
4. **ml-specialist.pdf** - ML Specialist Certificate (when completed)

## How to add certificates:

1. Download your certificate PDFs from Codecademy
2. Rename them to match the filenames above
3. Place them in this `/public/certificates/` folder

The certificates will open in a new tab when users click on the certificate cards on the Home page.
